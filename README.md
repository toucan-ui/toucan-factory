# Toucan Wizard

A visual theme configurator for [Toucan UI](https://github.com/toucan-ui/toucan-factory). Lets you customise design tokens — colours, typography, spacing, radii — and export a ready-to-use theme.

## Tech stack

- **Framework:** Next.js 16
- **Components:** Toucan UI core + patterns
- **Export:** Compressed theme output via fflate
- **Analytics:** Vercel Analytics

## Local development

```bash
# Install dependencies
pnpm install

# Start dev server (port 3002)
pnpm dev
```
