# Toucan Docs

The documentation site for [Toucan UI](https://github.com/toucan-ui/toucan-factory) — a token-driven design system.

## Tech stack

- **Framework:** Next.js 16
- **Content:** MDX
- **Styling:** Toucan UI tokens and components
- **Analytics:** Vercel Analytics

## Local development

```bash
# Install dependencies
pnpm install

# Start dev server (port 3001)
pnpm dev
```
